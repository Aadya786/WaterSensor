## WaterSensor

Welcome to the **WaterSensor** project! This repository contains everything you need to build a custom water/soil moisture sensing system with a live visual readout.

---

## 🛠️ Required Parts List

To build this project, you will need the following components:

* **Microcontroller:** [ESP32 Development Board (38-Pin Narrow Version)](https://www.amazon.com/KeeYees-Development-Bluetooth-Microcontroller-ESP-WROOM-32/dp/B07QCP2451/) - A 2.4 GHz Dual Core WLAN WiFi + Bluetooth microcontroller.
* **Prototyping:** [Solderless Half Breadboard & Flexible Jumper Wires Kit](https://www.amazon.com/124-Pcs-Breadboard-Jumper-Wires/dp/B0GVD11LSS/) - Perfect for students, beginners, and DIY circuit prototyping.
* **Display:** [0.96 Inch OLED Module (128x64 Yellow/Blue)](https://www.amazon.com/UCTRONICS-SSD1306-Self-Luminous-Display-Raspberry/dp/B072Q2X2LL/) - An I2C serial self-luminous display board powered by the SSD1306 driver.
* **Sensor:** [Capacitive Soil Moisture Sensor](https://www.amazon.com/dp/B0DYDN9RG4) - A corrosion-resistant moisture sensor built with the TLC555I chip.

---

## Next Steps
1. Connect your ESP32 to the breadboard.
2. Wire up the Capacitive Moisture Sensor to an analog pin.
3. Wire the OLED display via the I2C pins (SDA/SCL).
4. Upload your code and start tracking!
